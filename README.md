# GlobalStore - Excel Module Analysis
An interactive Excel dashboard analyzing GlobalStore sales, customers, products, and regional performance through dynamic charts and KPIs.

## License
This project is for educational purposes. Feel free to use and adapt the dataset for learning and analysis.

## Overview
This project consists of an interactive Excel dashboard built from the GlobalStore dataset. The objective was to analyze sales performance, customer patterns, product profitability, and regional trends through dynamic visualizations and automated calculations.

## Objectives
- Clean and structure the raw GlobalStore dataset.
- Create calculated fields to support KPI measurement.
- Build an interactive dashboard with slicers and filters.

## Tools & Techniques
- Microsoft Excel
- Pivot Tables and Pivot Charts
- Slicers
- Data cleaning and transformation in Excel
- Interactive Dashboard
